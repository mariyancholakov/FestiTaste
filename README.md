# CullinaryFestivals-Events

FestiTaste Events is a Next.js website for culinary festivals and food events in Bulgaria.

## Tech Stack

- Next.js 16 App Router
- React 19
- TypeScript
- Tailwind CSS 4
- Framer Motion
- Lucide React

## Scripts

```bash
npm run dev
npm run lint
npm run build
```

## Pages

- `/` Home
- `/events` Events
- `/offers` Offers
- `/about` About
- `/contact` Contact
